# Battle of Dots 2

Second version of [Keruki2005 and my Battle of Dots](https://github.com/Keruki2005/battle-of-dots) created in Godot 4. Currently not a full game, still in early developement. 

To play or develop, clone the repository and import the project folder into Godot 4.6 unsing the Project Manager of Godot (the window that opens on Godot's startup).
